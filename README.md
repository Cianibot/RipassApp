Progetto RipassApp Scopo della pagina La pagina è uno strumento di ripasso ottimizzato per smartphone. Mostra domande organizzate in cards, con risposte nascoste che diventano visibili solo dopo un’interazione dell’utente.

Come usare la pagina Scorri le cards Leggi la domanda Tocca “Mostra risposta” per visualizzare la risposta Tocca “Nascondi risposta” per chiuderla Le risposte non sono mai visibili automaticamente.

Struttura La pagina contiene: un titolo; più cards, ognuna con: una domanda; un pulsante; una risposta nascosta.

Interazione L’interazione avviene tramite tap/click sul pulsante. La risposta appare con animazioni (slide e dissolvenza) per migliorare l’esperienza utente su mobile.

Tecnologie utilizzate HTML per la struttura CSS per grafica e animazioni JavaScript per l’interazione Non vengono usate librerie esterne.

Modifica dei contenuti Cambiare titolo: h1 Cambiare domanda: .question Cambiare risposta: .answer
